Project Overview

1. Server-side (ASP.NET/C#)

  • Use ASP.NET Core with MVC architecture.
  • Set up two models in the Azure database with a one-to-many relationship using Entity Framework Core (ORM).
  • Create controllers to manage requests for these models.

2. Client-side (Angular)

  • Build a Single Page Application (SPA) using Angular.
  • Use services to communicate with your ASP.NET API.

3. Third-Party Identity (Okta)

  • Integrate Okta for user authentication and authorization.
  • Secure your Angular application using Okta's Angular SDK.
